**Self-Managed Logging Aggregator**

This project is a centralized logging system that collects and 
manages logs from different applications in one place.
It helps developers monitor system activity, detect errors, and 
analyze logs easily through a single dashboard instead of checking multiple log files.

**Features**

• Collect logs from different applications
• View all logs in one dashboard
• Search and filter logs easily
• Monitor system activity in real time
• Helps in debugging and error detection

**Local Setup Guide**

**Step 1: Download the Project**

Open the GitHub repository → Click Code → Select Download ZIP.

**Step 2: Extract the File**

Go to Downloads → Extract the ZIP file → Open the extracted folder.

**Step 3: Open in VS Code**

Right-click inside the project folder → Click Open in Terminal → Type: code .

**Step 4: Open Terminal in VS Code**

Press: Ctrl + `
Or go to Terminal → New Terminal

**Step 5: Install Dependencies**

Run: npm i

**Step 6: Run the Project**

Run: npm run dev

**Step 7: Open the Website**

After running the command, a link like this will appear:

http://localhost:9002

Copy the link and open it in your browser.

Requirements

• Node.js must be installed
• Run npm i before starting the project
